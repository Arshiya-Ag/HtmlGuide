Input Elements

text Ekement

  • Element default input hast va ye box ke mitonim toye on text benevisim ro Dorost Mikonad.

button Element

  • In Element Yek Dokme Ra Be Shekl Zir Dorost Mikonad.

checkbox Element

  • In Element Yek checkbox Ra Be Vojod Miyavarad Ke Ma Mitonim Az In Baraye Gereftan Taeed Baraye Mavared Mokhtalef Estefade konim.

color Element

  • In Element Be Shoma Yek Color Picker Midahad Ke mitonim Ba on Yek Rang Ra Entekhab konim.

date Element

  • In Element Be Shoma Yek Abzar Baraye Entekhab yek Tarikh (year/month/day) Midahad.

datetime-local Element

  • In Element ALave Bar Ghableiat Vared Kardan Tarikh Be Ma Ghabeliat Vared Kardan Time Ham Midahad.

email Element

  • In Element Hamon Ghabeliyat Haye text ro Darad Ama email Daraye Ghabeliyat haye Khasi Hast.

file Element

  • In Element Be Shoma Ghabeliyat Entekhab Ye file Ro Dar Roye Pc Khod Midahad.

image Element

  • Ba In Element Mitonim Yek Aks ro Be Jaye button Gharar Dahim.

month Element

  • In Element Faghat Sal Va Mah ra Darad (year/month).

number Element

  • Ba Estefade Azin Tag mitonim Yek Adad Ra Entekhab konim Be Sorat Zir.

password Element

  • Ba Estefade Azin tag yek Boxi Ra Be Vojod Miyarim Ke Dar On Harche Benevisim ***** Be in sorat Mishavad.

radio Element

  • Ba Estefade Az Chand ta Azin ha mitonim Yek halat soal chand gozine e be vojod biyarim.

  •     

range Element

  • Yek Bar Ro Be Vojod Miare Ke Ma Mitonim Yek Range ro Dar On Tanzim konim.

reset Element

  • Ba Zadan In Dokme Harche Ke to form hast Be Halat Defeult bar migardad.

search Element

  • Be Ma Ghabeliat Search Kardan Ro Midahad.

submit Element

  • Yek Dokme Baraye Taeed Form Baraye ma Be Vojod Miyavarad.

tel Element

  • Yek Ghesmat ro Baraye Vared Kardan Shomare Dorost Mikonad Va Dar Bazi Az browser Ha Yek Keypad Ham Hamrahash miayad.

time Element

  • In Element Be Ma Yek Ghesmati Ro Midahad Baraye Vared Kardan Time Bedon Yek Time Zone Moshakhas.

url Element

  • Dar In Elememt Ma Mitonim Url gharar Dahim Va Ba Inke url Benazar Shabihe text Hast Ama Yek Mahdodiat Ha Va Parametr Haye Makhsosi Darad
